Juego de Damas

Selecciona un color para tu tablero

1. El primer turno es para las fichas claras (rojas).
2. Sólo se permiten movimiento es diagonal sobre los cuadros oscuros.
3. No está permitido colocar dos fichas en el mismo recuadro.
4. Se puede comer a una ficha contraria pasando sobre esta hasta el recuadro siguiente.
5. Puede comer de manera consecutiva.
6. las fichas solo pueden avanzar hacia el frente (no pueden retroceder).
7. Sólo las reinas pueden retroceder.
8. Para convertirse en reina debe llegar a la primera fila del bando contrario (fila 1 para las fichas oscuras y fila 8 para las fichas claras)
